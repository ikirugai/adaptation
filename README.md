# Adaptation

React SDK for [Adaptation](https://adapt.ikirugai.com) — a drop-in Adapt button that lets users reshape your app in plain English. Popular requests graduate into defaults for everyone.

This repo holds the two public npm packages. Install them and read each package's README for usage.

```bash
npm install @ikirugai/adapt-react @ikirugai/adapt-server
```

- **[@ikirugai/adapt-react](https://www.npmjs.com/package/@ikirugai/adapt-react)** — drop-in React components (Adapt FAB, drawer, kanban, snip overlay).
- **[@ikirugai/adapt-server](https://www.npmjs.com/package/@ikirugai/adapt-server)** — `defineSurface`, session helpers, route-handler factories.

## See it live

[adapt.ikirugai.com](https://adapt.ikirugai.com) — four demo apps using this SDK. Click the Adapt button on any of them.
